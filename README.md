

# TCS Interns Project

Spring Boot REST API for managing Projects and Mentors.

## Features
- Create Project
- Get Project
- Update Project
- Delete Project
- Mentor mapping
- Exception Handling
- JUnit + Mockito
- Code Coverage

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- JUnit 5
- Mockito

## APIs
POST   /tcsinterns/project  
GET    /tcsinterns/project/{id}  
PUT    /tcsinterns/project/{id}  
DELETE /tcsinterns/project/{id}
